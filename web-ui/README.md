# Mobile Legends RAG Web UI

Web interface untuk Mobile Legends Knowledge Graph RAG system.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Axios

## Setup

1. Install dependencies:
```bash
cd web-ui
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open browser: `http://localhost:3000`

## Mobile Legends Theme
- Primary Color: Deep Blue (#2B4C7E)
- Accent Color: Gold/Yellow (#FDB933)
- Dark Background: (#0F1419)
- Responsive design with ML-inspired styling
